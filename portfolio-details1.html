Portfolio Detail

Halaman Home
Halaman ini menampilkan seluruh products berdasarkan products terbaru dan jumlah penjualan products terbanyak. Pada halaman ini dilengkapi dengan pagination yang menampilkan setiap 3 products.

Halaman Login & Register
agar user dapat masuk kedalam aplikasi ketika melakukan order.

Halaman Shop
Halaman ini menampilkan seluruh products yang tersedia, pada halaman ini terdapat filter yang memungkinkan user dapat memilih products berdasarkan (Harga, Kategori, Rating, Brand, Warna, Pengiriman)

Halaman FIlter Products
Pada halaman ini user akan melihat products berdasarkan dari kategori product yang dipilih oleh user maupun berdasarkan dari sub kategori dari product itu sendiri.

Halaman View Product
Halaman ini menampilkan rincian dari product yang dipilih oleh user mulai dari (Title, price, category, sub categories, shipping, color, brand, ketersedian barang, dan jumlah stock). Pada halaman ini juga ditampilkan carousel gambar dari product yang membuat user lebih mengenal product yang dijual.

Halaman View Product
Pada halaman ini juga ditampilkan product yang berkaitan dengan product view yang dipilih oleh user, sehingga membuat user lebih leluasa memilih product yang sesuai kebutuhan.

Halaman View Product
Pada halaman ini juga ditampilkan rating yang akan diberikan oleh user berdasarkan kepuadan terhadap product yang di lihat oleh user.

Komponen Drawer
Bagian ini akan muncul ketika user menambahkan product ke dalam keranjang, pada bagian ini user akan melihat item product yang terdapat pada keranjang/cart.

Halaman Cart
Pada halaman ini terdapat berbagai item product yang dipilih oleh user sebelum checkout. Pada halaman ini user dapat mengubah product yang dipilih berdasarkan warna serta jumlah product yang diinginkan. Pada bagian ini juga terdapat order summary dari product yang berada di dalam halaman ini. Dan juga user dapat memilih metode pembayaran yang dinginkan baik melalui kartu kredit (Proceed to checkout) maupun Cash on delivery.

Halaman checkout
Pada halaman ini user diminta memasukkan alamat tujuan pengiriman product, sebelum melakukan proses pembayaran. Pada bagian ini user juga dapat memasukkan kupon diskon untuk mendapatkan potongan harga dari product yang akan di beli. Untuk pembayaran secara cash on delivery user akan di arahkan ke halaman User History. Pada bagain ini terdapat action Empty Cart untuk mengosongkan seluruh cart user.

Halaman Payment
Ketika user memilih pembayaran dengan kartu kredit, user akan di arahkan pada halaman ini untuk menginputkan nomor kredit card untuk melakukan proses pembayaran.

Halaman User History
Pada halaman ini berisi riwayat order/pesanan yang dilakukan oleh user. pada bagian ini juga user dapat meilhat status pesanan apakah telah diproses atau tidak. Pada bagian ini juga user dapat menyimpan file pdf dari invoice orderan.

Halaman Password
Pada halaman ini user dapat mengubah password lama dengan password baru dengan memasukkan password yang lama. setelah itu user akan diberikan link untuk mengupdate password melalui email yang digunakan.

Halaman Wishlist
Pada halaman ini berisi whislist dari product yang dipilih oleh user, dari title product tersebut jika user mengklik akan dia rahkan ke halman view product secara rinci

Halaman Dashboard Admin
Pada halaman ini berisi seluruh orderan dari yang terbaru sampai lama. Pada halaman ini Admin dapat memilih apakah orderan tersebut akan (diproses, tidak diproses, dikirim, dibatakan, dan selesai).

Halaman Admin Dashboard (Product)
Pada halaman ini admin dapat menambahakan product yang akan dipublish ke website ecommerce

Halaman Admin Dasboard (Products)
Pada halaman ini berisi seluruh products yang dipublish oleh admin, pada bagian ini admin dapat melaukan tindakan berupa menghapus products maupun mengupdate products.

Halaman Admin Dashboard (Category)
Pada halaman ini admin dapat membuat category dari product yang akan dipublish ke website ecommerce. Pada bagian ini admin dapat melakukan tindakan berupa menghapus category maupun mengupdate category yang telah tersedia. Pada bagian ini juga tersedia fitur searching yang memudahkan admin untuk mencari category yang sesuai dengan product.

Halaman Admin Dashboard (Sub Category)
Pada halaman ini admin dapat membuat sub category berdarkan dari kategori utama dari product. Pada bagian ini admin dapat melakukan tindakan berupa menghapus sub category maupun mengupdate category yang telah tersedia. Pada bagian ini juga tersedia fitur searching yang memudahkan admin untuk mencari sub category yang sesuai dengan product.

Halaman Admin Dashboard (Coupon)
Pada Halaman ini admin dapat membuat Coupon untuk mendapat potohan harga yang bisa dijadlkan pad ahri hari tertentu atau even-evnet menarik.

Header
Bagian ini merupakan header pada website ecommerce ini terdiri dari navigasi Home, Shop, Cart, Login, Register dan fitur searching Products.

Project information

  • Category: Web
  • Project date: 26 juny, 2022
  • Live Demo: E-commerce Mern
  • Project URL: Github

Website e-commerce ini dibuat dengan menerapkan teknologi MERN yaitu React JS untuk frontend, redux untuk state management, Node JS dan Express JS untuk backend dan MongoDB sebagai database. Hak akses yang diterapkan dalam website e-commerce ini dibagi ke dalam 3 role yaitu sebagai admin, user dan guest. Untuk role admin sudah disediakan halaman dashboard tersendiri yang dapat memudahkan untuk mengelola website e-commerce. Sedangkan guest memiliki hak akses yang sangat terbatas karena ada beberapa fitur website yang hanya bisa diakses jika sudah login (role user). Adapun fitur-fitur di dalam website e-commerce ini akan dijelaskan secara detail sebagai berikut
