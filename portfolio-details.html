Page Home menampilkan semua Item yang dapat difilter berdasarkan dari kategori (drinks, rice, noodles, snack, pastry) yang dipilih oleh user. Secara default menampilkan semua item yang ada.

Page Login & Register agar user dapat masuk kedalam aplikasi ketika melakukan order.

Page Cart berguna menampilkan seluruh item yang di pilih oleh user sebelum melakukan order. Terdapat action yang dapat dilakukan oleh user berupa manambah jumlah item yang dipilh dan menghapus item yang dipilih

Page Invoice untuk memasukkan data user berupa (nama, kontak, alamat, serta metode pembayaran) ketika melakukan order

Page Bills menampilkan keseluruhan list orderan dari user/customer

Page Invoice action dari page bills untuk dapat mencetak/print orderan dari user/customer

Page Customer menampilkan keseluruhan list customer yang terdaftar pada aplikasi/sistem berupa (id, nama, dan kontak) dari user

Page Item pada page ini dapat melakukan action berupa menambah, update, melihat dan menghapus item yang berada dalam aplikasi/sistem

Project information

  • Category: Web
  • Project date: 26 juny, 2022
  • Project URL: github.com/pos-stackMern

Aplikasi Point of Sale merupakan sebuah aplikasi yang dirancang khusus untuk mempermudah dan mempercepat proses transaksi penjualan. Dirancang menggunakan framework ReactJs dengan design yang elegan menjadi daya tarik aplikasi ini. Selain itu, basic design dari aplikasi ini menggunakan Bootstrap dan And Design sehingga memudahkan web developer untuk mengembangkan design aplikasi ini menjadi lebih menarik lagi.
Proses transaksi penjualan dan pembelian menggunakan AJAX (Asyncronous JavaScript and XML). Memungkinkan user berkomunikasi dengan database tanpa reload page.
Adapun fitur dari aplikasi ini sebagai berikut:

  1. Management Data Barang
  2. Management User(Pengguna)
  3. Transaksi Pembelian
  4. Laporan order
